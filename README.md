# Plataforma Móvil de Telemetría para Pozos (aplicativo-movil)

Este proyecto es una aplicación web **Mobile First y Responsive** para visualizar la telemetría de pozos de agua y el cumplimiento normativo de la DGA (Dirección General de Aguas, Chile).

## Estructura del Proyecto

El proyecto está dividido en:

- **frontend/**: Aplicación de Next.js (App Router, TypeScript, Tailwind CSS, ECharts y MapLibre).
- **backend/**: API REST desarrollada con FastAPI y SQLAlchemy/Pydantic para el consumo de analíticas.
- **.claude/**: Skills y directrices para el desarrollo guiado por agentes de IA.

Para más detalles, por favor consulta [CLAUDE.md](CLAUDE.md).

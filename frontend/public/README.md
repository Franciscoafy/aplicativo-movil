# Carpeta de Recursos Públicos (Assets)

Coloca aquí los archivos de imagen, logos y otros recursos estáticos de la aplicación.

Para usar el logo de la marca **Quantica**:
1. Guarda el archivo del logo en esta carpeta con el nombre `logo.png` (o `logo.svg`).
2. La aplicación buscará el logo automáticamente en la ruta `/logo.png` (o `/logo.svg`).
